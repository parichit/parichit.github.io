Put your images and CV in this folder:

  portrait.jpg    — your headshot (square works best)
  cv.pdf          — your CV / résumé (the Download CV buttons link here)
  research-1.jpg  — optional project thumbnails (research-2.jpg, research-3.jpg…)
  og.jpg          — optional 1200×630 social-share preview image

Until you add them, the site shows tasteful placeholders, so nothing looks broken.
File names are referenced in content.js — keep them matching (or update content.js).
